# Функциональные требования — Fitness App

> Продуктовое описание основных пользовательских сценариев.

---

## 1. Общение с ИИ-тренером

### Ценность

Пользователь может задать любой вопрос о тренировках, питании, восстановлении и получить компетентный ответ на естественном языке. ИИ-ассистент выступает в роли персонального тренера, который всегда под рукой.

### Функциональные возможности

- **Свободная беседа** — пользователь пишет вопрос произвольной формы, система возвращает развёрнутый ответ с практическими рекомендациями
- **Контекст профиля** — ответы адаптированы под пол, возраст, вес, уровень подготовки и цель пользователя (если профиль заполнен)
- **Учёт травм и противопоказаний** — при наличии данных система учитывает ограничения и не рекомендует опасные упражнения
- **Сохранение истории** — все сообщения сохраняются, пользователь может вернуться к предыдущим беседам
- **Несколько сессий** — можно вести несколько независимых чатов на разные темы

### Флоу пользователя

1. Пользователь открывает чат
2. Система создаёт новую сессию в режиме свободного общения
3. Пользователь задаёт вопрос (например: «Как похудеть?», «Сколько белка нужно?», «Болит колено после приседа»)
4. ИИ-ассистент формирует ответ с учётом профиля пользователя
5. Пользователь может продолжить диалог — контекст предыдущих сообщений сохраняется в рамках сессии
6. При необходимости пользователь может переключиться в режим создания тренировки (и обратно) без потери чата

### Адаптивность

- При отсутствии связи с ИИ-сервисом система автоматически переключается на встроенную базу знаний (22 экспертные статьи по ключевым темам)
- Если профиль пользователя не заполнен — ассистент даёт общие рекомендации и может предложить заполнить профиль для более точных советов

## 2. Создание тренировки через ИИ-ассистента

### Ценность

Вместо ручного подбора упражнений пользователь описывает свои цели и условия в формате диалога — система автоматически генерирует оптимальную тренировку или недельный план с учётом уровня подготовки, оборудования, доступного времени и физиологических особенностей.

### 2.1. Одиночная тренировка

#### Флоу пользователя

1. Пользователь выбирает создание тренировки
2. Указывает параметры:
   - **Цель** — сила, рост мышц, выносливость, похудение, общее здоровье, реабилитация, мобильность, рост ягодиц, рекомпозиция (9 вариантов)
   - **Уровень подготовки** — новичок, средний, продвинутый
   - **Фокус-мышцы** — грудь, спина, ноги, плечи, руки, кор или всё тело (опционально)
   - **Оборудование** — выбор из пресетов или вручную
   - **Длительность** — от 20 до 120 минут или автоматически
3. Система генерирует тренировку:
   - Подбирает упражнения с учётом цели, уровня, оборудования
   - Определяет количество подходов и повторений
   - Рассчитывает отдых между подходами
   - Формирует метрики (тоннаж, калории, усталость по мышцам)
4. Пользователь получает готовую тренировку и может начать её выполнение

#### Адаптивность

- **Пол** — для женщин усиливается акцент на ягодицы и заднюю поверхность бедра; для мужчин — на грудь, плечи и широчайшие
- **Возраст** — после 50 лет снижается объём, увеличивается отдых, исключаются высокоударные упражнения; до 25 лет объём выше
- **Индекс массы тела** — при ожирении снижается нагрузка упражнений с собственным весом; при дефиците массы — ограничиваются ударные нагрузки
- **История тренировок** — упражнения, которые часто использовались в последнее время, заменяются альтернативами для разнообразия
- **Усталость мышц** — перетренированные мышечные группы получают пониженный приоритет
- **Противопоказания** — запрещённые упражнения полностью исключаются, нерекомендуемые — получают штраф

### 2.2. Недельный план (через диалог)

#### Флоу пользователя

1. Пользователь начинает диалог с ИИ-ассистентом
2. ИИ пошагово собирает параметры:
   - **Тип плана** — одна тренировка или план на неделю
   - **Цель** — 9 вариантов
   - **Уровень подготовки**
   - **Фокус-мышцы** (для одиночной) / **Частота** (для недельного)
   - **Оборудование** — пресет или ручной выбор
   - **Дни недели** и **длительность** — можно выбрать автоматически
   - **Расширенные настройки** (опционально) — тип сплита, уровень активности, предпочтения по кардио, базовым движениям
3. Шаги, на которые пользователь уже ответил в профиле, пропускаются автоматически
4. После завершения диалога система:
   - Определяет оптимальный сплит (Full Body / Upper-Lower / PPL) на основе частоты и уровня
   - Распределяет мышечные группы по дням с учётом восстановления
   - Генерирует тренировку для каждого дня
   - Учитывает кросс-сессионный объём — мышцы не перегружаются в рамках недели
   - Создаёт готовый план с расписанием
5. Пользователь активирует план — система создаёт тренировочные сессии на 4 недели

#### Маршрутизация диалога

- Если пользователь выбрал «Рекомендуемые настройки» — расширенные шаги пропускаются
- Если «Настроить вручную» — система показывает дополнительные шаги в зависимости от цели:
  - Похудение / выносливость → выбор типа кардио
  - Сила / рост ягодиц → выбор приоритетных базовых движений
  - Выносливость → тип выносливости (мышечная / кардио / смешанная)
  - Похудение → целевой вес для отслеживания прогресса

## 3. Прохождение тренировки

### Ценность

Пользователь получает готовую тренировку с запланированными весами, подходами и повторениями — ему остаётся только выполнять. Система автоматически ведёт прогрессию нагрузок и продвигает тренировочный план.

### Флоу пользователя

1. Пользователь открывает домашний экран и видит сегодняшнюю тренировку (тип, описание, количество упражнений)
2. Открывает тренировку — видит список упражнений с запланированными подходами:
   - Разминочные подходы (для базовых упражнений)
   - Рабочие подходы с рекомендуемым весом и количеством повторений
   - Кардио-упражнения с запланированной длительностью и дистанцией
3. Выполняет каждый подход, вводя фактические данные:
   - Вес (кг)
   - Количество повторений
   - Оценка усилия RPE (опционально)
   - Длительность и дистанция (для кардио)
4. После завершения всех подходов отмечает тренировку как выполненную
5. Система автоматически:
   - Сохраняет результаты
   - Продвигает тренировочный план — планирует следующую тренировку
   - ИИ-система подбора весов рассчитывает нагрузки для следующей сессии

### ИИ-система подбора весов

Для каждого упражнения в каждой новой тренировочной сессии система прогнозирует рабочий вес:

- **Анализ истории** — загружаются последние 5 выполненных подходов для данного упражнения
- **Оценка максимального результата** — по формуле e1RM (estimated one-rep max) = вес × (1 + повторения / 30)
- **Прогрессивная перегрузка** — если все подходы были лёгкими (RPE ≤ 7), вес увеличивается; если тяжёлыми (RPE ≥ 9) — снижается
- **Корректировка по усталости** — при высокой нагрузке на мышечную группу за последние 48 часов вес снижается
- **Начальный вес** — при отсутствии истории определяется по типу движения, уровню подготовки и ИМТ

### Разминочные подходы

Для базовых упражнений (присед, жим, тяга) с рабочим весом более 30 кг автоматически планируются:
- 1-й подход: пустой гриф × 15 повторений
- 2-й подход: 50% рабочего веса × 10 повторений
- 3-й подход: 75% рабочего веса × 8 повторений

### Пропуск тренировки

- Пользователь может пропустить тренировку с опциональным переносом на другой день
- Система автоматически находит ближайший свободный день и создаёт дубликат сессии
- Запланированные тренировки, не выполненные вовремя, автоматически помечаются как пропущенные в конце дня

### Продвижение плана

- После выполнения тренировки система определяет следующий тренировочный день
- Если неделя завершена — создаётся новая неделя сессий из шаблонов
- После 4 недель план автоматически архивируется

## 4. Планирование тренировок

### Ценность

Пользователь создаёт структурированный многомесячный тренировочный план с расписанием по дням недели. План автоматически генерирует сессии, ведёт прогрессию и управляет восстановлением.

### Флоу пользователя

#### Создание плана

1. Пользователь создаёт план — вручную или через ИИ-генерацию (п. 2.2)
2. При ручном создании:
   - Указывает название
   - Назначает шаблон тренировки на каждый день недели
   - Настраивает время тренировок
3. При ИИ-генерации план создаётся автоматически на основе диалога с заполненным расписанием

#### Активация

1. Пользователь активирует план
2. Система:
   - Деактивирует предыдущий активный план (если есть)
   - Создаёт активную сессию плана на 4 недели
   - Генерирует тренировочные сессии на каждый запланированный день
   - ИИ-система подбора весов рассчитывает нагрузки для ближайшей тренировки
3. Только один план может быть активным одновременно

#### Жизненный цикл

1. Пользователь выполняет тренировки по расписанию
2. После каждой тренировки система автоматически планирует следующую
3. По завершении недели — создаётся новая неделя из шаблонов
4. Через 4 недели план архивируется автоматически
5. Пользователь может архивировать план досрочно

#### Управление

- **Редактирование расписания** — активный план можно редактировать: при изменении расписания старые запланированные сессии пересоздаются
- **Замена тренировки** — конкретную запланированную сессию можно заменить на другой шаблон
- **Архивация** — план деактивируется, выполненные данные сохраняются
- **Удаление** — только неактивные планы можно удалить

### Домашний экран

На главном экране пользователь видит:
- Активный план (название, текущая неделя)
- Расписание на неделю с статусами (запланирована / выполнена / пропущена)
- Сегодняшнюю тренировку с описанием и количеством упражнений
- Тип каждой сессии переводится в понятное описание (например: «push» → «Грудь + плечи + трицепс»)
